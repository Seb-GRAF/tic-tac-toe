# Tic Tac Toe

Tic Tac Toe, great way to train Javascript

[LIVE](https://Seb-GRAF.github.io/tic-tac-toe)

# Features

- Player vs player mode (local)
- AI mode with a difficulty button (easy or hard)
- Mobile friendly

# AI

Implemented a minimax algorithm for the AI as part of a "hard" difficulty mode
